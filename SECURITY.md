# Security Policy

CuteBlock is a portfolio browser extension project. Please do not include private browsing data, tokens, cookies, or screenshots with personal information in public issues.

## Reporting

If you find a security issue, open a GitHub issue with a minimal reproduction that does not expose private data.

Useful details:

- Browser and version.
- Extension version or commit.
- The page pattern involved, using a sanitized example if needed.
- Whether optional online image sources were enabled.

## Scope

In scope:

- Permission model concerns.
- Content-script behavior that affects pages unexpectedly.
- Privacy issues related to optional external requests.
- Reversible replacement or site pause failures.

Out of scope:

- Requests to bypass publisher paywalls or access controls.
- Reports that require sharing private browsing history or account data.
